Implementación C++ para el sistema Arduino - Radio FM Inteligente

🔧 Arduino C++ - Radio FM System
#include <iostream> #include <string> using namespace std; class RadioFM { private: double frecuencia; string modo; bool encendido; public: RadioFM() { frecuencia = 100.1; // Frecuencia inicial modo = "FM"; encendido = true; } void subirFrecuencia() { frecuencia += 0.1; if (frecuencia > 108.0) frecuencia = 88.0; mostrarPantalla(); } void bajarFrecuencia() { frecuencia -= 0.1; if (frecuencia < 88.0) frecuencia = 108.0; mostrarPantalla(); } void cambiarModo() { if (modo == "FM") modo = "AM"; else if (modo == "AM") modo = "USB"; else if (modo == "USB") modo = "SD"; else modo = "FM"; mostrarPantalla(); } void mostrarPantalla() { cout << "Modo: " << modo << " | Frecuencia: " << frecuencia << " MHz" << endl; } void reproducirAudio() { if (modo == "USB" || modo == "SD") { cout << "Reproduciendo audio desde " << modo << "..." << endl; } else { cout << "Escuchando señal " << modo << " a " << frecuencia << " MHz" << endl; } } }; int main() { RadioFM radio; radio.mostrarPantalla(); // Simulación de botones radio.subirFrecuencia(); radio.subirFrecuencia(); radio.bajarFrecuencia(); radio.cambiarModo(); radio.reproducirAudio(); return 0; }

🔧 Funciones del Sistema

⬆️ subirFrecuencia()

Incrementa la frecuencia en 0.1 MHz. Al superar 108.0 MHz, reinicia automáticamente en 88.0 MHz.

⬇️ bajarFrecuencia()

Disminuye la frecuencia en 0.1 MHz. Al bajar de 88.0 MHz, salta automáticamente a 108.0 MHz.

🔄 cambiarModo()

Cicla entre modos: FM → AM → USB → SD → FM en rotación continua e intuitiva.

📺 mostrarPantalla()

Actualiza la pantalla LED mostrando el modo actual y la frecuencia en tiempo real.

🔊 reproducirAudio()

Gestiona la reproducción inteligente desde USB/SD o sintonización de frecuencias FM/AM.

🏗️ Constructor RadioFM()

Inicializa el sistema: frecuencia 100.1 MHz, modo FM predeterminado y estado activo.

📋 Especificaciones Técnicas

  • 🎛️ Control de frecuencia: Navegación precisa con incrementos de 0.1 MHz
  • 📡 Modos múltiples: Soporte FM, AM, USB y reproducción desde tarjeta SD
  • 🔄 Sistema cíclico: Navegación automática sin límites de rango
  • 📺 Display integrado: Visualización en tiempo real del estado del sistema
  • 🔊 Audio inteligente: Adaptación automática según el modo seleccionado
  • ⚙️ Arquitectura OOP: Código modular y escalable optimizado para Arduino
  • 🎯 Testing integrado: Simulación completa de todas las funciones del sistema
